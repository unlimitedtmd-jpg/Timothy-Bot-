## 🛠️ Jinsi ya Kudeploy (Hatua kwa Hatua)

1. **Unda Akaunti ya Heroku:** Hakikisha una akaunti kwenye [Heroku](https://www.heroku.com/).
2. **Bonyeza Kitufe:** Bonyeza kitufe cha `Deploy to Heroku` hapo juu.
3. **Jaza Variables (Config Vars):** Weka API Keys, Tokens, na mipangilio mingine inayohitajika kwenye fomu itakayotokea.
4. **Deploy App:** Bonyeza **"Deploy App"** chini kabisa ya ukurasa na usubiri mchakato ukamilike.
5. **Washa Bot (Dynos):** Baada ya kumaliza, nenda kwenye tab ya `Resources` kwenye Heroku dashboard yako na uwashe (`turn on`) worker dyno.

## ⚙️ Mahitaji (Requirements)

Kabla ya kudeploy, hakikisha una vitu vifuatavyo (vya kuweka kwenye Config Vars):
- `API_KEY` (Kama ipo)
- `BOT_TOKEN` (Kutoka kwa BotFather au platform husika)
- `OWNER_ID` (ID yako)
